Streaming Documentation

Live TV Streams

Construct URLs using the pattern: /stream/{ChannelNumber}/{Format}/{Codec}

  • Format: ts (MPEG-TS), mp4 (Fragmented MP4), mkv (Matroska). (Defaults to ts if omitted, or mkv if codec is av1)
  • Codec: copy (Direct Stream), h264, h265, av1. (Defaults to server setting `TRANSCODE_CODEC`, usually h264)
http://{ServerIP}:3000/stream/5.1/mp4/h264
http://{ServerIP}:3000/stream/15.2/ts/copy

M3U Playlist

Generate a playlist for your media player using query parameters.

  • f (Format): ts, mp4, mkv
  • c (Codec): copy, h264, h265, av1
http://{ServerIP}:3000/playlist.m3u?f=mp4&c=h264
